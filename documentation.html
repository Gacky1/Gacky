Getting Started with Gacky

Learn how to set up and configure Gacky for your Discord server.

Installation

  • Click the "Add to Server" button on our website
  • Select your Discord server from the dropdown
  • Grant the necessary permissions for full functionality
  • Run ghelp to see all available commands

Basic Configuration

  • Set your server prefix: gsetprefix <new_prefix>
  • Configure moderation settings: gautomod status
  • Set up logging channels: gmodlog #channel
  • Configure welcome messages: gwelcome setup

Essential Commands

Access comprehensive command documentation

gping

Check bot latency and database status

gbotstats

View bot statistics and information

ginvite

Change server prefix (max 2 characters)

Moderation Commands

Powerful moderation tools to keep your server safe and organized.

gselfclean

Auto cleanup messages from the bot

gclear [amount]

Delete multiple messages at once

gkick @user [reason]

Kick a user from the server

gban @user [reason]

Ban a user from the server

gunban @user

Unban a previously banned user

grole @user @role

Add a role to a user

grrole @user @role

Remove a role from a user

glock

Lock the current channel

gunlock

Unlock the current channel

gmaintenance

Enable server maintenance mode
