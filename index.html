Features

Moderation

  • role, rrole - Role management
  • lock, unlock - Channel control
  • maintenance - Server maintenance mode

Esports

  • smanager - Slot management
  • tourney - Tournament system
  • idp - ID Password for room
  • easytag, tagcheck - Tag management
